Support:

* zen-cart-v1.3.9h
* Zen Cart v1.5.1/v1.5.1


Requirements:

* Swedish_language

* Curl


The modul don't support localhost or local ip-address.


Installation:

* You should have a backup of your web store and database

* You should install the Swedish_language before installing the module

* Download the module “payson_1.8_for_zencart” on your PC

* extract the module and go to the folder payson_1.8_for_zencart/ 

* Copy all the files in the zencart folder to the root directory of your store 


Configuration:

* Login into your web shop Administration Panel

* Go to Modules->Payment and install Payson Nytt API 

* Enter your Email, Agent ID and MD5 Key before enabling the module

* Go to Modules->Payment and install Payson Faktura Nytt API

* Enter your Email, Agent ID and MD5 Key before enabling the module

* Go to Modules->Order Total and install Fakturaavgift

* Enter Faktureringsavgift before enabling the module


Upgradation: 

* From payson_zencart_XX to payson_1.8_for_zencart 

* You should have a backup of your web store and database

* Login into your web shop Administration Panel

* Go to Modules->Payment and Uninstall Payson Nytt API 

* Go to Modules->Payment and Uninstall Payson Faktura Nytt API

* Go to Modules->Order Total and Uninstall Fakturaavgift

* Download the module "payson_1.8_for_zencart" on your PC

* Extract the module and go to the folder payson_1.8_for_zencart

* Go to your web shop  and rename the folder 
zencart/includes/modules/payment/payson to zencart/includes/modules/payment/payson_old

* Copy the folder payson_1.8_for_zencart/includes/modules/payment/payson to the folder  zencart/includes/modules/payment/  of your web shop.

* Go to your web shop  and rename the file 
zencart/includes/modules/payment/payson.php to zencart/includes/modules/payment/payson.php_old

* Copy the file payson_1.8_for_zencart/includes/modules/payment/payson.php to the folder  zencart/includes/modules/payment/  of your web shop.

You have now upgraded your module successfully.
